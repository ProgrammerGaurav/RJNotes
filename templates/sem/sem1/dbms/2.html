Deleting a Column in a table
Code: 
            
                alter table student
                drop column roll_no;
                select * from student;
Output: 
            
                +----+---------+
                | id | Name    |
                +----+---------+
                |  1 | Gaurav  |
                |  2 | Harshad |
                +----+---------+
Adding a Column in a table
Code: 
            
                alter table student
                add column roll_no int;
                select * from student;
Output: 
            
                +----+---------+---------+
                | id | Name    | roll_no |
                +----+---------+---------+
                |  1 | Gaurav  |    NULL |
                |  2 | Harshad |    NULL |
                +----+---------+---------+
Renaming a table
Code: 
            
                rename table student to students;
                show tables;
Output: 
            
                +-------------------+
                | Tables_in_student |
                +-------------------+
                | students          |
                +-------------------+
Truncating a table (Deleting all the content of a table)
Code: 
            
                truncate table students;
                select * from students;
Output: 
            
                Empty set (0.00 sec)
Dropping a table (Deleting a table)
Code: 
            
                drop table students;
                select * from students;
Output: 
            
                ERROR 1146 (42S02): Table 'student.students' doesn't exist
