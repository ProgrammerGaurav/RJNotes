Python List
List is a collection which is ordered and changeable. Allows duplicate members.
Code: 
            
                # Creating a list
                thislist = ["apple", "banana", "cherry", "orange", "kiwi", "melon", "mango"]

                # Adding Element to list
                thislist.append("pineapple")
                print(thislist)

                # Removing Element to list
                thislist.remove("apple")
                print(thislist)

                # Printing a List
                print(thislist)

                # Printing 2nd element of list
                print(thislist[1])

                # Printing Last element of list
                print(thislist[-1])

                # Return the third, fourth, and fifth item:
                print(thislist[2:5])

                # This example returns the items from the beginning to "kiwi":
                print(thislist[:4])

                # This example returns the items from the "kiwi" to endling:
                print(thislist[3:])
Output: 
            
                ['apple', 'banana', 'cherry', 'orange', 'kiwi', 'melon', 'mango', 'pineapple']
                ['banana', 'cherry', 'orange', 'kiwi', 'melon', 'mango', 'pineapple']
                ['banana', 'cherry', 'orange', 'kiwi', 'melon', 'mango', 'pineapple']
                cherry
                pineapple
                ['orange', 'kiwi', 'melon']
                ['banana', 'cherry', 'orange', 'kiwi']
                ['kiwi', 'melon', 'mango', 'pineapple']
Python Tuple
List is a collection which is ordered and changeable. Allows duplicate members. Python Tuple is same as list except it is immutable(uneditable) A tuple is an immutable sequence of Python objects. Tuples are sequences, just like lists. The differences between tuples and lists are, the tuples cannot be changed unlike lists and tuples use parentheses, whereas lists use square brackets. You can perform every list operation on tuple except operations releated to adding and removing
