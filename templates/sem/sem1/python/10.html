Classes in Python
It is a collection of object and member function which access that object. It is declared with keyboard class followed by class name.
Code: 
            
                # Creating a class
                class student:
                    # Initialization of values
                    def __init__(self, name, rollno):
                        self.name = name
                        self.rollno = rollno
                
                    # Function to display name and rollno
                    def displayinfo(self):
                        print("Name : ", self.name)
                        print("Roll No. : ", self.rollno)
                
                # creating two object of class student
                student1 = student("Gaurav", 26)
                student2 = student("Harshad", 89)
                
                student1.displayinfo()
                student2.displayinfo()
Output: 
            
                Name :  Gaurav
                Roll No. :  26
                Name :  Harshad
                Roll No. :  89
