Math module
This module provides access to the mathematical functions defined by the C standard. It consist of pre-defined functions which we can use for calculations in our program.
Code: 
            
                # Import math module
                from math import *

                a = 4

                # pow to calculate power
                print(pow(a, 2))

                # sqrt for square root
                print(sqrt(a))

                # factorial to calculate factorial
                print(factorial(a))
Output: 
            
                16.0
                2.0
                24
Random module
This module implements pseudo-random number generators for various distributions.
Code: 
            
                # Import Random module
                from random import *

                # randint function returns random numbers between the parameters
                print(randint(1, 6))

                # choice Return a random element from the non-empty sequence seq. If seq is empty, raises IndexError.
                a = [1, 2, 3, 4, 5, 6]
                print(choice(a))
Output: 
            
                6
                3
